Sensory Toy Guide - About

Find the best sensory toys for people with autism, ADHD, and sensory processing disorder.


Note - This page contains a lot of development infomation.



Language Infomation

The Sensory Toy Guide is written in HTML, CSS, JavaScript and Ruby. It is built with Jekyll, and pushed to GitHub pages with an actions runner.


Contribution Infomation

If you would like to contribute, feel free! Your local development machine will need Ruby and Jekyll for development. Once your contribution is ready, please make a PR detailing your changes.


Why was this site created?

This site was created because I personally seen normal toys sold and marketed as sensory toys. I have seen Tomodatchi-like devices being sold as sensory toys, so I decided to make a site to help people when looking for sensory toys.
